Welcome to the UML2Cloud Eclipse plugin update site.

This web page provides automatic distribution and updates for the Eclipse plugin for UML2CLoud.

Plugin requirements

This plugin has primarily been tested with Eclipse Oxygen 3, but let us know if you have any problems. The plugin runs under Java 1.8, or newer. This plugin requires at least Papyrus 3.0.0.

Plugin installation

To install the UML2Cloud plugin:

  1. In Eclipse, click on Help -> Install Software...
  2. Click Add.
  3. Enter the following:
    • Name: UML2Cloud
    • URL: http://antares.sip.ucm.es/cana/UML2Cloud/
    and click OK.
  4. Click the checkbox next to "UML2Cloud" it to select it, and click Finish.
    Note: Disable Group items by category option.
    Note: Enable Contact all update sites during install to find required software option.
  5. Click Next.
  6. Select the I accept the terms of the license agreement option to accept the license and click Next.
  7. The plugin is not digitally signed. Go ahead and install it anyway clicking Ok.
  8. Click Yes to make Eclipse restart itself.
